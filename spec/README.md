# specscore-cli specification

This directory contains the specification for the `specscore` CLI — the reference implementation of [SpecScore](https://specscore.md) tooling.

The format being implemented is specified separately at [synchestra-io/specscore](https://github.com/synchestra-io/specscore). This tree specifies only the CLI's contract — its commands, flags, output, exit codes, and behavior.

## Structure

```
spec/
└── features/
    └── cli/        # The specscore CLI feature tree
```

See [features/](features/README.md).
